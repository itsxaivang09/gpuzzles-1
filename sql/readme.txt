
[1] Download QuizMaster.zip and deploy into xampp/htdocs/QuizMaster

[2] Launch XAMPP; Start Apache Server and SQL Server; 

[3] Go to SQL Admin panel (phpmyadmin); Create a database called "quiz_master".

[4] Import the SQL file (xampp/htdocs/quiz_master/sql/quiz_master.sql) into quiz_master database.

[5] Launch the quiz_master application
http://localhost/quiz_master
